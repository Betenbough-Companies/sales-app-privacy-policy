# Sales App Privacy Policy

This repository contains the privacy policy for the Betenbough Homes internal sales application.

## GitHub Pages

The privacy policy is available on GitHub Pages. Once enabled, it will be accessible at:
`https://betenbough-companies.github.io/sales-app-privacy-policy/`

## Local Preview

To preview the page locally:
```bash
# Install Jekyll (if not already installed)
gem install jekyll bundler

# Serve the site locally
jekyll serve
```

Then visit `http://localhost:4000` in your browser.
